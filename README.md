# 项目介绍

这是一款极其简单的内容展示类小程序，它使用Markdown文档作为内容源，使用towxml插件将Markdown文档解析并展示到小程序。

目前该项目记录了与编程开发相关的知识点和经验，是学习开发小程序的好工具！

小程序版本更新内容参考Version.md 文件。

项目打开方法：

下载安装微信开发者工具，然后打开项目的mini文件夹即可（请不要选择云开发和小程序模板）。
